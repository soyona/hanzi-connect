# Project Brief - 汉字偏旁部首连连看游戏 (V1.0)

## 1. 核心价值与目标受众
* **核心价值**：通过“偏旁”与“剩余部件”的拼字连连看消除游戏，强化小学生（一至三年级）对汉字“模块化组合”的认知，降低形近字混淆率，寓教于乐[cite: 1]。
* **目标受众**：小学中低年级学生、教师及家长[cite: 1]。

## 2. 核心功能范围 (Scope)
* **关卡矩阵**：支持 4x4 或 6x6 卡牌矩阵，游戏开始时随机打散分配[cite: 1]。
* **卡牌拼字消除**：支持选择“左偏旁/上部首”与“右部件/下部件”进行组合，匹配教育部小学常用字库则消除并得分[cite: 1]。
* **游戏状态控制**：具备倒计时/生命值扣除机制、错误重置逻辑、全消除过关判定[cite: 1]。
* **本地数据持久化**：关卡进度、历史最高分完全持久化于浏览器 `LocalStorage`[cite: 1]。

## 3. 绝对黑名单 (Anti-Scope / Constraints)
* 🚫 **严禁生僻字**：必须严格基于“教育部小学常用字库”，禁止拼出任何小学阶段不涉及的生僻、古汉语汉字[cite: 1]。
* 🚫 **拒绝后端依赖**：无全球排行榜、无用户注册/登录系统、无真实后端数据库接入[cite: 1]。
* 🚫 **拒绝复杂特效**：采用纯前端 DOM 渲染与 Tailwind CSS，动画与音效最简化，聚焦于汉字匹配核心逻辑[cite: 1]。
